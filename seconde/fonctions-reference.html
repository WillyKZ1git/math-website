Fonctions de référence

Découvre les quatre grandes familles de fonctions que tu croiseras tout au long de ta scolarité : la fonction affine, la fonction carré, la fonction inverse et la fonction racine carrée.

Introduction

Définition

Une fonction est une correspondance qui associe à chaque réel x d'un ensemble de départ (le domaine de définition) un unique réel noté f(x).

En Seconde, le programme distingue quatre fonctions de référence : la fonction affine, la fonction carré, la fonction inverse et la fonction racine carrée. Connaître leur expression algébrique, leur courbe représentative et leurs propriétés est indispensable pour toute la suite du lycée.

1 Fonction affine

Définition

La fonction affine est définie sur ℝ par :

f(x) = ax + b

où a est le coefficient directeur (ou pente) et b est l'ordonnée à l'origine.

Lorsque b = 0, on parle de fonction linéaire : f(x) = ax

Propriétés
  • Sa représentation graphique est une droite.
  • Si a > 0 : la fonction est strictement croissante sur ℝ.
  • Si a < 0 : la fonction est strictement décroissante sur ℝ.
  • Si a = 0 : la fonction est constante (droite horizontale).
  • La droite coupe l'axe des ordonnées en (0 ; b).
  • La droite coupe l'axe des abscisses en (−b/a ; 0) si a ≠ 0.
Exemple

Soit f(x) = 2x − 3.

  • Coefficient directeur : a = 2 → la droite monte de 2 pour 1 vers la droite.
  • Ordonnée à l'origine : b = −3 → la droite coupe l'axe des ordonnées en (0 ; −3).
  • f est croissante sur ℝ.
  • f(x) = 0 ⟺ 2x − 3 = 0 ⟺ x = 3/2 : la droite coupe l'axe des abscisses en (1,5 ; 0).

2 Fonction carré

Définition

La fonction carré est définie sur ℝ par :

f(x) = x²

Propriétés
  • Sa courbe représentative est une parabole de sommet O(0 ; 0).
  • Elle est paire : f(−x) = f(x) → symétrie par rapport à l'axe des ordonnées.
  • Décroissante sur ]−∞ ; 0] et croissante sur [0 ; +∞[.
  • Minimum en x = 0 : f(0) = 0.
  • f(x) ≥ 0 pour tout réel x (toujours positive ou nulle).
Exemple

f(3) = 9  |  f(−3) = 9  |  f(0) = 0

f(x) = 4 ⟺ x² = 4 ⟺ x = 2 ou x = −2.

⚠ Attention

√(x²) = |x| et non x. Par exemple : √(9) = 3 ≠ −3.

3 Fonction inverse

Définition

La fonction inverse est définie sur ℝ \ {0} par :

f(x) = 1/x

Son domaine de définition est D = ℝ \ {0} car on ne peut pas diviser par zéro.

Propriétés
  • Sa courbe représentative est une hyperbole à deux branches.
  • Elle est impaire : f(−x) = −f(x) → symétrie par rapport à l'origine O.
  • Strictement décroissante sur ]−∞ ; 0[ et sur ]0 ; +∞[.
  • L'axe des abscisses et l'axe des ordonnées sont des asymptotes de la courbe.
  • La courbe est dans les quadrants 1 et 3 (signes identiques pour x et f(x)).
Exemple

f(2) = 0,5  |  f(−4) = −0,25  |  f(1) = 1

Quand x → +∞, f(x) → 0 (la courbe se rapproche de l'axe des abscisses sans jamais le toucher).

4 Fonction racine carrée

Définition

La fonction racine carrée est définie sur [0 ; +∞[ par :

f(x) = √x

Son domaine de définition est D = [0 ; +∞[ car on ne peut pas prendre la racine carrée d'un nombre négatif dans ℝ.

Propriétés
  • Strictement croissante sur [0 ; +∞[.
  • f(x) ≥ 0 pour tout x ≥ 0.
  • f(0) = 0 et f(1) = 1 : la courbe passe par (0 ; 0) et (1 ; 1).
  • (√x)² = x pour tout x ≥ 0.
  • √(x²) = |x| (et non x si x est négatif).
  • √(a × b) = √a × √b pour a, b ≥ 0.
Exemple

f(0) = 0  |  f(1) = 1  |  f(4) = 2  |  f(9) = 3  |  f(25) = 5

Résoudre √x = 3 : x = 3² = 9.

Simplifier √12 = √(4 × 3) = 2√3.

Synthèse comparative

Fonction affine
f(x) = ax + b
  • Droite
  • Domaine : ℝ
  • Croissante si a > 0
Fonction carré
f(x) = x²
  • Parabole
  • Domaine : ℝ
  • Min en x = 0
Fonction inverse
f(x) = 1/x
  • Hyperbole
  • Domaine : ℝ \ {0}
  • Décroissante sur chaque intervalle
Racine carrée
f(x) = √x
  • Demi-parabole
  • Domaine : [0 ; +∞[
  • Croissante
Exercice 1 Identifier une fonction affine Facile

Parmi les expressions suivantes, lesquelles définissent une fonction affine ? Pour chacune, précise le coefficient directeur a et l'ordonnée à l'origine b.

  1. f(x) = 3x + 5
  2. g(x) = x²
  3. h(x) = −2x
  4. k(x) = 7
  5. m(x) = x/2 − 1
Correction

f(x) = 3x + 5 ✓ — Affine avec a = 3, b = 5.

g(x) = x² ✗ — Ce n'est pas une fonction affine (c'est la fonction carré).

h(x) = −2x ✓ — Affine (linéaire) avec a = −2, b = 0.

k(x) = 7 ✓ — Affine constante avec a = 0, b = 7.

m(x) = x/2 − 1 ✓ — Affine avec a = 0,5, b = −1.

Exercice 2 Calculer des images Facile

Calcule les images suivantes :

  1. f(x) = x² ; calculer f(−5), f(0), f(√3).
  2. g(x) = 1/x ; calculer g(4), g(−2), g(1/3).
  3. h(x) = √x ; calculer h(0), h(16), h(1/4).
Correction

f(−5) = (−5)² = 25  |  f(0) = 0  |  f(√3) = (√3)² = 3

g(4) = 1/4 = 0,25  |  g(−2) = 1/(−2) = −0,5  |  g(1/3) = 1/(1/3) = 3

h(0) = 0  |  h(16) = √16 = 4  |  h(1/4) = √(1/4) = 1/2

Exercice 3 Sens de variation Intermédiaire

Sans calculatrice, compare les valeurs suivantes en utilisant les variations des fonctions de référence :

  1. √7 et √5 (sans calculer les valeurs)
  2. 1/3 et 1/5
  3. (−3)² et (−4)²
  4. √0,25 et 0,25
Correction

√7 > √5 — la fonction racine carrée est croissante sur [0 ; +∞[ et 7 > 5.

1/3 > 1/5 — la fonction inverse est décroissante sur ]0 ; +∞[ et 3 < 5, donc 1/3 > 1/5.

(−4)² > (−3)² — c'est-à-dire 16 > 9. La fonction carrée est croissante sur [0 ; +∞[, et |−4| > |−3|.

√0,25 = 0,5 > 0,25 — pour x ∈ ]0 ; 1[, √x > x (car √x · √x = x et √x > 0 implique √x > x quand √x < 1).

Exercice 4 Résoudre des équations Intermédiaire

Résoudre dans ℝ les équations suivantes. Préciser le domaine de définition avant de résoudre.

  1. x² = 16
  2. √x = 5
  3. 1/x = −2
  4. x² = −4
Correction

x² = 16 — domaine ℝ. x² = 16 ⟺ |x| = 4 ⟺ x = 4 ou x = −4. S = {−4 ; 4}

√x = 5 — domaine [0 ; +∞[. √x = 5 ⟺ x = 25 (car 5 ≥ 0). S = {25}

1/x = −2 — domaine ℝ \ {0}. 1/x = −2 ⟺ x = 1/(−2) = −1/2. S = {−1/2}

x² = −4 — domaine ℝ. Or x² ≥ 0 pour tout réel x, donc x² = −4 est impossible dans ℝ. S = ∅

Exercice 5 Domaine de définition Intermédiaire

Déterminer le domaine de définition maximal (dans ℝ) de chacune des fonctions suivantes, puis la classer parmi les fonctions de référence.

  1. f(x) = √(x − 3)
  2. g(x) = 1/(x + 2)
  3. h(x) = (x − 1)²
Correction

f(x) = √(x − 3) — il faut x − 3 ≥ 0, soit x ≥ 3. Domaine : [3 ; +∞[. C'est une fonction racine carrée décalée.

g(x) = 1/(x + 2) — il faut x + 2 ≠ 0, soit x ≠ −2. Domaine : ℝ \ {−2}. C'est une fonction inverse décalée.

h(x) = (x − 1)² — définie pour tout réel. Domaine : ℝ. C'est une fonction carré décalée, parabole de sommet (1 ; 0).

Exercice 6 Problème ouvert — Mise en situation Difficile

Un objet est lâché sans vitesse initiale depuis une hauteur h (en mètres). Sa hauteur au bout de t secondes est donnée par :

h(t) = 20 − 5t²

  1. Quelle est la hauteur initiale de l'objet ?
  2. Quelle est la hauteur après 1 seconde ? Après 2 secondes ?
  3. Au bout de combien de secondes l'objet touche-t-il le sol (h = 0) ? Donner une valeur exacte et une valeur approchée.
  4. De quelle fonction de référence la fonction h(t) se rapproche-t-elle (en ignorant les coefficients) ?
Correction

1. h(0) = 20 − 5 × 0² = 20 m. La hauteur initiale est 20 mètres.

2. h(1) = 20 − 5 = 15 m. h(2) = 20 − 5 × 4 = 20 − 20 = 0 m.

3. h(t) = 0 ⟺ 20 − 5t² = 0 ⟺ t² = 4 ⟺ t = 2 (on garde t > 0). L'objet touche le sol après t = 2 secondes exactement.

4. h(t) = 20 − 5t² est de la forme A − B·t² : c'est une fonction carré (avec coefficient et décalage). La parabole est tournée vers le bas (coefficient de t² négatif).
